Аутичные дети
Ниже приведен список рекомендаций для родителей аутичный детей:
  • Дети-аутисты болезненно переживают изменения в их обычном распорядке дня. Старайтесь поддерживать стабильность во всем, что касается аутиста.
  • Необходимо ежедневно проводить упражнения. На начальных этапах достаточно выделять 5-7 минут, но постепенно нужно увеличивать длительность до двух-трех часов. Не забывайте делать перерывы.
  • Научите ребенка-аутиста слову "Стоп". Оно крайне важно для него, так как он может не видеть реальную опасность, например, прикоснуться к кастрюле с кипящей водой. Выработать этот навык можно путем резких прерываний разных действий по слову "Стоп".
  • Ограничивайте время пребывания у телевизора и компьютера.
  • Для успешного развития ребенок должен общаться со сверстниками. Учтите, что сам аутист не предрасположен к долгому общению, поэтому рассмотрите вариант с посещением детского сада на несколько часов.
  • Прерывайте монотонные действия ребенка с помощью танцев и прыжков на месте.
  • Сделайте для ребенка какое-нибудь "тайное место", в котором он мог бы оставаться наедине с собой.
  • Качели вызывают особый интерес у детей-аутистов. Используйте их не только как средство для развлечение, но и для развития вестибулярного аппарата.
  • Если вы хотите завести домашнего питомца, предварительно подготовьте ребенка с помощью игрушки к общению с животным. Так как в поведении аутистов часто может быть немотивированная агрессия в сторону животных.
Список коррекционных методик:
1. Занятия с карточками.

Используйте карточки для объяснения выполнения или порядка выполнения определенных действий. Для начала достаточно пяти карточек, но со временем их количество должно увеличиваться. Алгоритм действий прост: вы произносите определенную команду и далее показываете карточки с порядком действий для выполнения этой команды. При этом каждая карточка должна сопровождаться кратким пояснением. Главное, регулярно повторять эти занятия и со временем ребенку будет достаточно лишь одной команды для выполнения действия. Карточки должны быть заготовлены на все основные действия: "гулять", "рисовать", "читать" и т.д.

2. Профилактика страха воды.

Дети-аутисты обычно боятся воды. Для профилактики данного страха нужно предлагать игры, в которых будет задействована вода. Сначала лучше начать с мокрого песка, например, рисовать на нем линии и делать разные фигуры.

3. Развивайте тактильное восприятие.

Можно заполнить небольшую коробочку песком и внутри спятать несколько пуговиц, для начала 2-3 штук будет достаточно. Задача ребенка найти их все. Со временем количество пуговиц можно увеличивать до 15.

4. Развитие координации.

Возьмите две чашки, одну из них заполните водой. Задача ребенка перелить воду во вторую чашку. По мере развития размер пустой чашки должен уменьшаться.

5. Занимайтесь лепкой.

Для лепки можете использовать массу для моделирования или пластилин. Эта методика одновременно будет развивать как моторику, так и сенсорную чувствительность.

6. Собирайте пазлы.

Для начала покупайте пазлы с небольшим количеством деталей, но при этом они должны быть крупными. По мере развития размер деталей должен уменьшаться, а их количество увеличиваться.

7. Занимайтесь танцами.

Можно танцевать дома или в каком-нибудь кружке.

Самое важное – принимайте ребенка таким, какой он есть. Он не лучше и не хуже других детей, он просто другой…
